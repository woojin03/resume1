CallGuard
CallGuard는 보이스피싱 예방을 위한 애플리케이션으로, 사용자가 보이스피싱 위험으로부터 안전할 수 있도록 다양한 기능을 제공합니다. 이 프로젝트는 Python과 Java를 기반으로 개발되었으며, 주요 기능으로는 실시간 통화 음성 분석, 보이스피싱 위험 알림, 번호 신고 및 조회 등이 있습니다. Oracle DB를 사용하여 데이터를 관리하고, ai 음성을 학습한 콜가드 자체 알고리즘이 실시간으로 위험을 감지합니다. CallGuard는 보이스피싱의 위협으로부터 사용자를 보호하고 안전한 통신 환경을 제공하는 것을 목표로 하며, 팀원들의 협업과 기술적 도전을 통해 완성되었습니다.

AI 음성 데이터 비교 문제
문제 인식
AI 모델에서 생성된 음성 데이터와 실제 사용자 음성을 비교하는 데 있어 기준과 방법론이 명확하지 않아 분석 결과의 신뢰성 문제가 제기되었습니다. 이로 인해 효과적인 비교 방법을 찾아야 했습니다.
문제 해결
다양한 음성 데이터 비교 방법을 조사하고, 관련 논문을 참고하여 음성 유사도 비교를 위한 알고리즘을 적용했습니다. 이를 통해 AI 음성 패턴과 실제 데이터를 효과적으로 비교할 수 있는 기준을 확립하여 문제를 해결할 수 있었습니다.
분석 위치 결정 문제
문제 인식
통화 분석 기능을 서버에서 처리할지, 앱에서 직접 처리할지 결정하는 데 있어 성능, 사용자 경험, 네트워크 부하 측면에서 최적의 방안을 선택하기 어려웠습니다.
문제 해결
K-해커톤 멘토링 세션에서 전문가의 피드백을 받아 앱에서 직접 분석을 수행하는 방안이 사용자 경험을 개선하고 네트워크 부하를 줄이는 데 효과적이라는 결론을 얻었습니다. 이에 따라 분석 로직을 앱에서 실행하도록 설계하고 구현을 수정하여 문제를 해결했습니다.
